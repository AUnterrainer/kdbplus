
This code was developed using below reference.

1. Shimko, David C(1993): "Bounds of Probability", Risk, 6, 33-7

Pre-requisite: 

1. Have q installed. To install q, go to www.kx.com.
2. You also need to download stat.q file. Can be downloaded from http://kx.com/q/stat.q
3. Implied Volatility Curve. Here I use Nikkei 225 Options data available from Osaka Stock Exchange (OSE) 
website. 

To run this:

1.Run irndSvc.q

        >q irndSvc.q
        
2.Open irnd.html in your web browser.
