
An estimation of physical distribution based on historical data using Gaussian Kernel.

Pre-requisite: Have q installed. To install q, go to www.kx.com.
You also need to download stat.q file. Can be downloaded from http://kx.com/q/stat.q

To run this:

1.Run gaussian.q

        >q gaussian.q
        
2.Open gaussian.html in your web browser.
